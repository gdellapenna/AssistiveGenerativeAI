The University of L'Aquila website includes accessibility features to support users with visual or reading impairments. These features are part of an accessibility section that details the portal's structure and adherence to accessibility standards, such as separating content from presentation and providing flexible design according to W3C guidelines.

Specifically, the site offers:
- A "Dichiarazione di accessibilità" (Accessibility Declaration) that confirms evaluation of the site's accessibility and compliance with legal requirements.
- Quick access keys (keyboard shortcuts) designed to facilitate navigation for users who prefer or need to use the keyboard instead of a mouse.
- A feedback mechanism to report accessibility issues directly to the university’s accessibility officer.

However, the information does not explicitly mention options like a high contrast mode or a font size selector on the pages, so these might not be directly available as toggle features.

***

How to access accessibility features with a screen reader:
- Look for a link or section labeled "Accessibilità" or "Accessibility" usually found in the footer or main menu.
- Use Tab or heading navigation to find information about quick access keys and the accessibility declaration.
- If available, shortcut keys allow direct keyboard navigation to important sections to enhance accessibility.

Therefore, while the site has a strong commitment to accessibility and keyboard navigation support, there is no clear indication of a dedicated user-controlled high contrast or font size selection feature on the University of L'Aquila website as of now.[1][2]
