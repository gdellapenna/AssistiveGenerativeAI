Yes, the Wikipedia Main Page includes several menus and navigation areas. Here’s a breakdown of all menu options based strictly on the current page content:

---

## 🖥️ Visual Menu Overview

### 1. **Top Navigation Bar (Header)**
Located at the very top of the page:

- **Main page** – Returns to the homepage.
- **Talk** – Discussion page for the Main Page.
- **Read** – View the page content.
- **View source** – See the page’s source code.
- **History** – View edit history.

### 2. **Sidebar Menu (Left Side)**
This vertical menu appears on the left side of the screen:

#### **Navigation**
- Main page
- Contents
- Current events
- Random article
- About Wikipedia
- Contact us
- Donate

#### **Contribute**
- Help
- Learn to edit
- Community portal
- Recent changes
- Upload file

#### **Tools**
- What links here
- Related changes
- Special pages
- Permanent link
- Page information
- Wikidata item

#### **Print/export**
- Download as PDF
- Printable version

#### **Languages**
- Links to versions of the page in other languages

## 🗣️ Screen Reader Menu Guide

To access these menus:

### 1. **Top Navigation Bar**
- Use Tab to move through the top links:
  - “Main page,” “Talk,” “Read,” “View source,” “History”
- Each is a link. Press `Enter` to activate.

### 2. **Sidebar Menu**
- Use heading navigation (`H`) or Tab to reach the sidebar.
- The sidebar is structured into labeled groups:
  - “Navigation,” “Contribute,” “Tools,” “Print/export,” “Languages”
- Use arrow keys to explore each group.
- Use Tab to move between links and `Enter` to activate.

### 3. **Quick Link Listing**
- Press `Insert+F7` (JAWS/NVDA) to list all links on the page.
- This will include all menu items and content links.
