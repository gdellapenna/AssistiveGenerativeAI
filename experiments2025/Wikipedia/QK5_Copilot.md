To switch the Wikipedia Main Page to another language, follow these steps based on the current page layout:

---

## 🖥️ Visual Instructions

### Step-by-step:
1. **Look at the left-hand sidebar** of the page.
2. Scroll down to the **bottom section labeled “Languages.”**
3. You’ll see a list of language names (e.g., Español, Français, Deutsch).
4. **Click on the language** you want — this will take you to the Main Page in that language.

---

## 🗣️ Screen Reader Instructions

### Step-by-step:
1. Press `H` repeatedly until you reach the **heading “Languages.”**
2. Arrow down to hear a list of available languages.
   - Each language is a link (e.g., “Deutsch,” “Español,” “Français”).
3. Use the arrow keys to navigate through the list.
4. Press `Enter` on the language you want to switch to.

Alternatively, you can:
- Press `Insert+F7` (JAWS/NVDA) to open the links list.
- Type the first few letters of the language (e.g., “ita” for Italiano).
- Select the desired language and press `Enter`.
