There are navigation elements that function like menus or primary links, primarily located at the very top of the page.

Here is a description of the menu options and how to access them.

### **Menu Options and Description**

#### **Visual Directions (General User)**

1.  Look at the top-left corner of the page for the navigation links.
2.  The first block of options, often presented horizontally or vertically at the start, are: **Home**, **Random**, and **Nearby**.
3.  The second block of options, usually on the right side of the header or beneath the first block, includes: **Log in**, **Settings**, and **Donate Now**.
4.  Below these, you'll see a link for **About Wikipedia** and another for **Disclaimers**.

#### **Screen Reader Directions (Blind User)**

1.  When you first land on the page, the screen reader should encounter these links almost immediately after the page title.
2.  **Use the "Next Link" command** (often **Tab** or **L**) starting from the very top of the main content area to cycle through the menu options.

***

| Link Text (Menu Option) | Screen Reader Landmark/Context | Description of Content |
| :--- | :--- | :--- |
| **Home** | Primary navigation link | Navigates to the Main Page (the page you are currently on). |
| **Random** | Primary navigation link | Navigates to a random article in the encyclopedia. |
| **Nearby** | Primary navigation link | Navigates to content related to your geographical location (if location services are enabled). |
| **Log in** | Utility link | Takes you to a page to sign in to an existing Wikipedia account. |
| **Settings** | Utility link | Takes you to a page to change your user preferences (requires an account/login). |
| **Donate Now** | Utility link | Takes you to a page to make a donation to Wikipedia. |
| **About Wikipedia** | Utility link (often near the bottom, too) | Takes you to a page providing information about Wikipedia itself. |
| **Disclaimers** | Utility link (often near the bottom, too) | Takes you to a page listing legal disclaimers about the content. |
| **Search** | Located in the search bar | Allows you to type in a topic to find a specific article. |

***

In summary, the page features **eight primary/utility links** (`Home`, `Random`, `Nearby`, `Log in`, `Settings`, `Donate Now`, `About Wikipedia`, `Disclaimers`) and one main **Search** feature.
