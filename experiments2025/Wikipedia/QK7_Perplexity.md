To search for a specific entry using the Wikipedia main page:

Visually:
- At the top right corner of the page header, there is a search box with placeholder text "Search Wikipedia."
- Click or tab into this box and type the word or phrase you want to find.
- After typing, press the Enter key or click the magnifying glass icon next to the input box to start the search.
- If your search exactly matches a page name, Wikipedia will take you directly to that page.
- Otherwise, it shows a list of search results containing your entered terms.

For screen reader users:
- Use quick navigation or landmark keys to find the "Search Wikipedia" edit box near the top of the page.
- Press Tab repeatedly until you reach the search input field.
- When focused in the search box, type the entry or keywords you want.
- After typing, press Enter to perform the search.
- If an exact page exists for your term, focus will move directly to that page’s content.
- If no exact match, you will be presented with search results to navigate through and select from.

This search box is standard across Wikipedia and allows you to quickly find any article or topic on the site from the main page.
