The Wikipedia main page has a few key menus, mainly arranged for navigation and easy access to different sections and features. Describing the menus as they appear and function:

Visually:
- There is a top horizontal menu bar with:
  - A "Main Page" link (usually the current page)
  - Links to "Contents," "Current events," "Random article," "About Wikipedia," and "Contact us"
- A sidebar menu on the left side of the page with these main options:
  - Navigation section including links such as "Main page," "Contents," "Current events," "Random article," "Donate to Wikipedia," and "Wikipedia store"
  - Interaction section with links like "Help," "About Wikipedia," "Community Portal," "Recent changes," and "Upload file"
  - Tools section including links to what you can do with the page, like "What links here," "Related changes," "Upload file," and "Special pages"
  - Print/export section with "Download as PDF," "Printable version"
  - Languages section offering links to Wikipedia in different languages

For screen reader users:
- The top horizontal menu is a landmark navigation bar with links for quick access to main site areas. You can navigate through it using landmark and link navigation keys.
- The left sidebar is a navigation landmark with clear section headings labeled "Navigation," "Interaction," "Tools," "Print/export," and "Languages."
- Under each heading, multiple link items are listed that take you to different Wiki sections or functions.
- You can navigate by heading level to reach each menu section quickly and then arrow through the links.
- These menu links are normally among the first things accessible when tabbing from the top page area.

The menus provide quick access to essential Wikipedia features like browsing contents, exploring current events, random articles, user help, community tools, file uploads, and language switching.
