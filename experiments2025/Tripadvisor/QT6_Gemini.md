This is a core function of the page! Searching for a specific hotel is done using the main search bar, but you must first ensure the search is set to the **"Hotels"** category.

Here is the step-by-step process:

### 1. Visual Directions (What you see):

1.  Locate the set of category tabs/buttons directly above the main search bar.
2.  Ensure the **"Hotels"** tab is selected. If it isn't, click on the **"Hotels"** tab.
3.  The large search box will now be prompted to **"Enter a destination or property name."**
4.  Click inside the search box and begin typing the name of the **specific hotel** you are looking for (e.g., "The Grand Plaza Hotel").
5.  As you type, a list of matching hotels and destinations might appear below the search box. Select the exact hotel from this list if it appears, or finish typing.
6.  Click the **Search button** (often labeled by a magnifying glass icon or simply "Search") to submit your request.

### 2. Screen Reader Directions (How to navigate):

1.  Navigate to the main search area by pressing **"Tab"** or using the **"F"** key (for form controls).
2.  Use the **left/right arrow keys** to cycle through the search category tabs. Listen for the option: **"Hotels."**
    * *Action:* Ensure the **"Hotels"** radio button/tab is selected. You should hear **"Hotels, selected"** or **"Hotels, checked."** If not, press the right or left arrow key until it is selected.
3.  Press **"Tab"** once to move into the main search text field.
    * *Listen for:* The screen reader will announce the field, something like **"Enter a destination or property name, edit text."**
4.  Type the full name of the hotel you are searching for into this text field.
5.  If a list of suggestions appears, use the **down arrow key** to navigate through the suggestions. Your screen reader will announce them. Press **"Enter"** when you hear the correct hotel name.
6.  If you don't use the suggestions, press **"Tab"** again to find the **"Search" button** and press **"Enter"** or **"Spacebar"** to submit your search.
