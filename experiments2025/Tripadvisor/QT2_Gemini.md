This is an excellent question for understanding the page's functionality!

On this page, you can search for four main categories of travel information: **Hotels, Things to Do, Restaurants, and Forums.**

The page provides a central search box that changes its focus depending on which category you select.

### 1. Visual Directions (What you see):

1.  Look for the main search bar in the center of the page, which usually says, **"Search all things travel."**
2.  Immediately above this search bar, there is a set of buttons (or tabs) that let you choose what you are searching for. These buttons are labeled:
    * **"Hotels"**
    * **"Things to Do"**
    * **"Restaurants"**
    * **"Forums"** (This is a less common section, but it is available for searching user discussions).
3.  The text in the search bar changes to guide you: for instance, if you click **"Hotels,"** the box will prompt you to **"Enter a destination or property name."**

### 2. Screen Reader Directions (How to navigate):

1.  Press **"Tab"** or use the **"F"** key (for form controls) to navigate to the main search area. You should hear the main search text field, possibly labeled **"Search all things travel, edit text."**
2.  Before this text field, there is a group of **radio buttons** or **tabs** that control the search category. You can use the **left/right arrow keys** to switch between these options. Listen for:
    * **"Hotels"**
    * **"Things to Do"**
    * **"Restaurants"**
    * **"Forums"**
3.  Select the category you want by stopping on it. For example, if you want to find something to do, navigate until you hear **"Things to Do, radio button checked/selected"** or **"Things to Do, selected tab."**
4.  Once the category is selected, press **"Tab"** again to move into the search text field, and your screen reader will announce the updated prompt, such as **"Search things to do, edit text,"** telling you exactly what you can type there.
