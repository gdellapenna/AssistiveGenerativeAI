On the Cambridge English Dictionary page, there is an option to select or switch between different dictionaries or language pairs for the search.

### How to select dictionary or language options:
- Near the search box or at the top of the page, there is usually a dropdown menu or a language selector.
- This menu lets you choose from various bilingual dictionaries (such as English–Spanish, English–Chinese, English–French, etc.) or monolingual English dictionary resources.
- Selecting a dictionary here changes the language pair or dictionary set used for lookup.
- For instance, if you want to search in English–Spanish, you would select that option before typing your word in the search box.

### Screen reader guidance:
- When the page loads, use quick navigation commands to locate the dropdown list or combobox labeled with something like “Select Dictionary” or “Choose Language.”
- Use arrow keys to open and navigate the dropdown options to find your desired dictionary or language pair.
- Select the appropriate dictionary by pressing Enter or Space.
- After this, move to the search input field to type your word and press Enter to search within the selected dictionary.

This feature provides flexible search options beyond just English to English, supporting many language combinations on the Cambridge Dictionary site.[1][11][12]
