You can locate the search bar on the page using its position relative to the main headings and site structure.

### Locating the Search Bar

**👁️ Visual Directions:**

1.  The search bar is located at the **top of the page**, immediately following the site logo and the main navigational links (**Dictionary**, **Translate**, **Grammar**, etc.).
2.  It is positioned directly under the large main heading that says **Cambridge English Dictionary: Meanings & Definitions**. The text input box is clearly visible and is the widest interactive element in that upper section of the page.

**🔊 Screen Reader Directions:**

1.  Use the landmark navigation command to jump directly to the **Search landmark**. This is the most efficient way to land near the input field and its associated button.
2.  Alternatively, you can navigate to the **Main landmark**. The search input will be one of the very first interactive elements encountered within the main content.
3.  You can also use the form element navigation command (e.g., pressing `E` or `F`) to quickly jump to the first **input field** on the page, which is the search bar itself.
