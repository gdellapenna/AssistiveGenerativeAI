The primary search function on this page, which is located at the top and labeled **English Dictionary**, automatically searches across three major Cambridge dictionaries simultaneously: the **Cambridge Advanced Learner's Dictionary, 4th Edition**, the **Cambridge Academic Content Dictionary**, and the **Cambridge Business English Dictionary**.

However, you can choose to look at definitions from specific sub-dictionaries or editions by using direct links found further down the page, which are part of a hidden menu structure.

### Accessing Specific Dictionary Options

**👁️ Visual Directions:**

1.  Look in the area where navigation options are listed, below the main site header.
2.  Look for a section that lists specific dictionary names for definitions, under a heading like **Choose a dictionary**.
3.  In this area, you will find direct links to use specific dictionaries:
    * **English Learner's Dictionary**
    * **Essential British English**
    * **Essential American English**
4.  Clicking on one of these links will load the page structure optimized for that specific dictionary, allowing you to use its search function.

**🔊 Screen Reader Directions:**

1.  Use the link navigation command (e.g., pressing `L`) or search for links labeled specifically with the dictionary titles.
2.  Look for and activate the following links:
    * **English Learner's Dictionary**
    * **Essential British English**
    * **Essential American English**
3.  Activating one of these links will reload the page for that dictionary type.
