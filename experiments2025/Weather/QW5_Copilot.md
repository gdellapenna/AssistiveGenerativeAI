To search for the weather forecast of a specific location on weather.com, follow these steps:

---

### 🖥️ For Sighted Users

1. **Locate the search bar** at the top of the page. It’s positioned in the upper-right corner, next to the site logo.
2. **Click inside the search field**. You’ll see placeholder text like “Search City or Zip Code”.
3. **Type the name of the location** (e.g., “Rome, Italy” or a ZIP code).
4. As you type, a dropdown list of matching locations will appear.
5. **Click on the correct location** from the list.
6. You’ll be taken to a new page showing the current and upcoming forecast for that location.

---

### 🎧 For Screen Reader Users

1. **Navigate to the top of the page** using your screen reader’s “top of document” command.
2. Use the **“next form field”** or **“tab”** command until you hear “Search City or Zip Code edit field”.
3. **Activate the edit field** and type the name of the location (e.g., “Rome, Italy”).
4. Use the **“down arrow”** to hear the autocomplete suggestions.
5. When you hear the correct location, **press Enter** to select it.
6. The page will load with the forecast for that location. Use **heading navigation** to explore the forecast details.
