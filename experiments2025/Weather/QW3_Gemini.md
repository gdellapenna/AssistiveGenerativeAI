The forecast for the next ten days is easily accessible through the **Main Forecast Menu** I described earlier.

Here is how you can access it:

***

## 🗓️ Accessing the 10-Day Forecast

### Visual Instructions

1.  Locate the main horizontal menu bar, which is situated directly beneath the area displaying the current weather conditions.
2.  Look for the link labeled **"10 Day"** within this menu. It is usually the third or fourth option in the list, after "Today" and "Hourly."
3.  **Click** the **"10 Day"** link. This will load the detailed 10-day forecast view for the currently selected location.

### Screen Reader Instructions

1.  Use the **"L" key** (or the command to jump between links) to navigate the links in the main content area, or use the **Tab key** repeatedly after the search box.
2.  Stop when your screen reader announces the link text: **"10 Day."**
3.  Press **Enter** or **Spacebar** while focused on the **"10 Day"** link to select it and go to the 10-day forecast page.

**Note:** If you are using the Tab key, you will likely encounter the links in this order: "Today," "Hourly," and then **"10 Day."**
