## 📡 Accessing Radar Maps

### Visual Instructions

1.  Look for the primary navigation menu located near the top of the page, which lists the different forecast types (like "Today," "Hourly," and "10 Day").
2.  Find the link labeled **"Radar."** It is typically the last option in this list.
3.  **Click** the **"Radar"** link to view the current precipitation and weather map.

### Screen Reader Instructions

1.  Use the **"L" key** (or the command to jump between links) to navigate through the links in the main content area, or use the **Tab key** repeatedly.
2.  Continue pressing the link navigation key or the Tab key until your screen reader announces the link text: **"Radar."**
3.  Press **Enter** or **Spacebar** while focused on the **"Radar"** link to access the interactive map view.
