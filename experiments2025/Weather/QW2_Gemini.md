## Are there menus on this page?

**Yes**, there are several distinct sets of navigation links (menus) on the page.

### 1. Main Forecast Menu (Primary Navigation)

This is a set of links directly related to the current location's forecast, typically located near the top of the main content.

#### Visual Instructions

Look right below the Search Bar and the current location information. You will see a horizontal list of options for viewing the forecast in different time frames.

#### Screen Reader Instructions

This group of links is usually announced as a **"Navigation"** landmark or a list of links following the current weather summary. You can use the **Tab key** to move through these options sequentially, or use the link navigation command (like the **"L" key**) to find them.

| Link Text | Destination/Purpose |
| :--- | :--- |
| **Today** | Detailed forecast for the current day. |
| **Hourly** | Forecast broken down hour-by-hour. |
| **10 Day** | The extended 10-day forecast. |
| **Weekend** | Forecast specifically for the upcoming weekend. |
| **Monthly** | Forecast trends for the current month. |
| **Radar** | A map view showing precipitation and weather patterns. |
