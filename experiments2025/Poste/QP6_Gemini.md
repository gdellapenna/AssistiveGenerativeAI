Yes, there are multiple ways to log in or access your personal area directly from this page.

### **1. Personal Area MyPoste (Prominent Section)**

This is the most visible log-in area on the page.

#### **Standard Visual Directions**

Look near the top of the main content area for a box labeled "**Area Personale MyPoste**." Inside this box, there is a clear button labeled:

* **Accedi** (Access/Login)

#### **Screen Reader Directions**

1.  Use the **H** key to navigate by headings and listen for "**Area Personale MyPoste**" (or the nearby heading, if it is not an `H2`).
2.  Use the **Tab** key to find the button labeled **"Accedi"**. Press **Enter** or **Space** to log in.
3.  Alternatively, you may find the text link "**Accedi e registrati da desktop**" (Access and register from desktop).

### **2. Header Utility Links (Top of the Page)**

There are quick-access links in the header, always available at the top of the page.

#### **Standard Visual Directions**

Look in the header area, near the main navigation links, for the link labeled:

* **Area personale** (Personal Area)

Clicking this link will likely lead to a login prompt or a panel.

#### **Screen Reader Directions**

1.  Navigate to the top of the page.
2.  Use the **Tab** key to move through the utility links, listening for the link labeled **"Area personale"**. Press **Enter** on it.

### **3. Footer Links (Bottom of the Page)**

The log-in option is also duplicated in the comprehensive footer.

#### **Standard Visual Directions**

Scroll all the way to the bottom of the page (the Footer). Look for the section titled "**Area personale**" (Personal Area). The first link listed here is:

* **Accedi** (Access/Login)

#### **Screen Reader Directions**

1.  Use the **Skip to page footer** link or navigate to the very bottom of the page.
2.  Use the **H** key or **Down Arrow** key to find the footer section labeled **"Area personale"**.
3.  Use the **Tab** key to find the link labeled **"Accedi"**. Press **Enter** on it.
