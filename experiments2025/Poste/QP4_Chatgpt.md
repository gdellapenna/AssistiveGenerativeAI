Good question. I checked the live version of the page at **[https://www.poste.it](https://www.poste.it)** — yes: the page *does* contain a search bar. ([Poste.it][1])

Here’s what you can search for — and how — from that bar:

---

## 🔎 What can you search for on the page

* The page has a general site‑search bar labelled **“Cerca nel sito”** (Search the site). ([Poste.it][1])
* In addition, there is a dedicated **“Cerca spedizioni”** (“Track shipments / Search shipments”) field: you can enter a shipment/tracking code to check the status of a package or registered mail. ([Poste.it][1])
* There is also a tool **“Cerca un Ufficio Postale o un Punto Poste”** (“Find a Post Office or a Punto Poste”) that lets you search the location of post offices or authorized points (e.g. lockers, retail‑partner points) to find the one closest to you. ([Poste.it][1])

---

## 🧑‍🦯 How to use these search bars with a screen reader / keyboard

* To use the **site‑search bar** (“Cerca nel sito”):

  1. Use your screen reader’s command to move to the search landmark or to find a “search” form control — you should hear something like “Search: Cerca nel sito”.
  2. Once focused, type the term you want to search (e.g. “raccomandata”, “conto BancoPosta” etc.).
  3. Press Enter to submit the search; the page will load results related to your query.

* To **track a shipment** using “Cerca spedizioni”:

  1. Navigate to the section labelled “Cerca spedizioni”. Your screen reader should announce something like “Text field: Inserisci codice spedizione”. ([Poste.it][1])
  2. Enter your parcel/letter tracking code (the alphanumeric code from your shipping receipt).
  3. Navigate to the button labelled “Avvia la ricerca” (“Start search”) and press Enter. This will load a page showing the status of your shipment (if valid).

* To **find a post office / Punto Poste**:

  1. Navigate to the link labelled “Cerca un Ufficio Postale o un Punto Poste”. ([Poste.it][1])
  2. Activate the link (press Enter). This should open a form or a map-based search where you can input an address or postal code (CAP).
  3. Enter your search data, then submit — the site will return a list or a map of post offices / Punto Poste locations near the address you entered.
