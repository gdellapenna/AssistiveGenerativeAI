Yes, the page contains multiple structured menus and navigation groups. The main options are organized in a **Header Menu (Mega Menu)**, a set of **Utility Menus** in the header, and comprehensive lists in the **Footer**.

### **1. Header Mega Menu: "Tutti i prodotti e servizi" (All products and services)**

This is the largest, most comprehensive menu on the page.

#### **Standard Visual Directions**

Locate the **"Menu"** button near the top of the page. Clicking this opens a large, dedicated panel titled "**Tutti i prodotti e servizi**" (All products and services). The panel then displays a list of main categories and sub-options.

#### **Screen Reader Directions**

1.  Navigate to the main header section (near the top of the page).
2.  Use the **Tab** key to find and activate the **"Menu"** button. Press **Enter** or **Space**.
3.  Once the menu opens, use the **Down Arrow** key or the **H** key to navigate through the main categories, which function as section headings.
