Yes — the current Poste Italiane home page (**[https://www.poste.it/](https://www.poste.it/)**) contains a search functionality, but there are actually **two distinct types of search areas**, depending on what you want to search for:

---

## **1. Shipment / Parcel Search (“Cerca spedizioni”)**

### Visual Description

* There is a labeled input field near the top or middle of the main content: **“Cerca spedizioni”**.
* A text box where you can **enter a tracking code**.
* A button next to it (or below it) to start the search — typically labeled **“Cerca”** or **“Avvia la ricerca”**.

### What you can search for

* You can enter a **parcel/shipment tracking number** issued by Poste Italiane.
* It will return the **current status of your shipment** (location, expected delivery, history).

---

## **2. Site-Wide Search (general search bar)**

### Visual Description

* At the very top of the page, in the header, there is a **magnifying glass icon** or a small input labeled **“Cerca”**.
* Clicking on the magnifying glass opens a text input where you can type a query.

### What you can search for

* This search is for **all Poste Italiane content**, such as:

  * Services (sending parcels, opening accounts, prepaid cards)
  * Post office locations and office hours
  * Information about financial or insurance products
  * SPID / digital services
  * News or announcements from Poste Italiane

---

## **Screen Reader Navigation**

1. **Parcel / shipment search**

   * Navigate to the heading or landmark labeled **“Cerca spedizioni”**.
   * Focus on the input box (screen reader should announce something like “Text input: Inserisci codice spedizione”).
   * Type your tracking number, then move to the **search button** and activate it.

2. **Site-wide search**

   * Navigate to the **top navigation landmark**, locate the magnifying glass icon labeled **“Cerca”**.
   * Activate it — the screen reader will announce a **search input**.
   * Type your query and press Enter.

---

So in short:

* **“Cerca spedizioni”** → search for your parcel tracking number.
* **Top search bar / magnifying glass** → search the Poste Italiane website for services, offices, news, etc.
