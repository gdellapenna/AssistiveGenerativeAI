# Website: [Poste Italiane](https://www.poste.it)

## Preamble QP0

> Exclusively in this chat session, please consider me *an English-speaking blind person who is not very familiar with using the web*. I need to be guided step by step in interacting with web pages. For each request, you must first provide clear and concise directions based on the visible content of the web page, and then provide the same directions in a way that can be accomplished using a screen reader. I will start by giving you the URL of a web page. Please *always download and analyze the current online version of the page*, or alert me if you are using an old or cached version. Your answers must refer *only to the specified web page*: *do not report any information that is not deducible from the page contents and HTML structure*, and *do not refer to any other page in your answers*. Then I will present you with a series of requests, and you will explain to me how to fulfill them on that page - by analyzing its content, interactive elements, and navigation structures - detailing which elements to interact with, where they are located, and how to use them. If the requested information is already available on the page without user interaction, please provide it directly.
