Yes, this page contains multiple search-like input fields. Based on the page's content, you can search for two main types of information: **Shipments** and **Post Offices/Punto Poste locations**.

These tools are located in the main content area of the page, beneath the main promotional carousels.

### **1. Search for Shipments (Tracking)**

#### **Standard Visual Directions**

Look for a section titled "**Cerca spedizioni**" (Search shipments). Directly beneath this title, there is an input field labeled "**Inserisci codice spedizione**" (Enter shipment code). You should type the tracking code into this box.

#### **Screen Reader Directions**

1.  Use the **H** key to navigate by headings, or the **Down Arrow** key to scan the content after the main promotional section.
2.  Listen for the section header, which will be announced as "**Cerca spedizioni**".
3.  Immediately following this, use the **F** key to jump to form fields, or the **Tab** key. You will land on an edit field labeled "**Inserisci codice spedizione**".
4.  Type the tracking code into this input field and press **Enter** to initiate the search.

### **2. Search for Post Offices and Appointments**

#### **Standard Visual Directions**

Look for a section titled "**Cerca un ufficio e prenota**" (Find an office and book an appointment). Below this title is a text box labeled "**Cerca un Ufficio Postale o un Punto Poste**" (Search a Post Office or a Punto Poste). This is where you enter the name or location you are looking for.

#### **Screen Reader Directions**

1.  Continue navigating the page structure using the **H** key or **Down Arrow** key, looking for the section header: "**Cerca un ufficio e prenota**".
2.  Use the **F** key or **Tab** key to find the corresponding input field, which is labeled "**Cerca un Ufficio Postale o un Punto Poste**".
3.  Enter the location or name you want to search for into this box.
4.  There is also another prompt below the input field: "**Scegli il servizio e prenota un appuntamento**" (Choose the service and book an appointment), which suggests this search is part of a workflow for booking.
