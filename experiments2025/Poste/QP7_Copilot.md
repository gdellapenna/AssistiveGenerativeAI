Yes — the Poste Italiane home page (**[https://www.poste.it/](https://www.poste.it/)**) provides a dedicated way to **track the status of a shipment**.

---

## **1. Visual Description of Shipment Tracking**

* There is a section labeled **“Cerca spedizioni”** (Search shipments).
* Inside this section:

  * **Text input field**: where you enter your **tracking code** (numerical or alphanumerical code provided when you shipped a package).
  * **Button**: usually labeled **“Cerca”** or **“Avvia la ricerca”** to submit the tracking request.
* You type your tracking code into the input field and click the button. The page will then show the **current status of the shipment**, including:

  * Location updates
  * Delivery status (in transit, delivered, exception)
  * Expected delivery date

---

## **2. Screen Reader Navigation to Track a Shipment**

1. Use **heading navigation** or **landmark navigation** to move to the heading **“Cerca spedizioni”**.
2. Your screen reader will announce a **text input** (e.g., “Text input: Inserisci codice spedizione”).
3. Type the **tracking code** provided with your shipment.
4. Move to the **search button** next to the input (announced as “Cerca” or “Avvia la ricerca”) and activate it.
5. The page will refresh or display the shipment’s tracking information. The screen reader will then read:

   * Shipment location updates
   * Delivery progress/status
   * Expected delivery time or any exceptions

---

* **How:** Enter your tracking code → click/search → read status updates.
