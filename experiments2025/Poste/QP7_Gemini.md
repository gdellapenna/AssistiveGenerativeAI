Yes, it is possible to track the status of a shipment directly from this page.

The functionality is located in a dedicated search area in the middle of the page.

### **1. Standard Visual Directions**

1.  Scroll down the page until you find the section titled **"Cerca spedizioni"** (Search shipments).
2.  You will see an input field labeled **"Inserisci codice spedizione"** (Enter shipment code).
3.  Type the tracking number (codice spedizione) into this box.
4.  After entering the code, click the **"Cerca spedizioni"** button or icon (which may be implied by the form structure, but the action is initiated by submitting the code).

### **2. Screen Reader Directions**

1.  Use the **H** key to navigate through the main headings of the page until you hear the section related to tracking, announced as **"Cerca spedizioni"**.
2.  Use the **F** key to jump to the next form field, or the **Tab** key to move to the input box. You should hear it announced as the edit field labeled **"Inserisci codice spedizione"**.
3.  Type the tracking number into this field.
4.  Press the **Enter** key while the input field is focused. This action will submit the code and take you to the shipment status results.
